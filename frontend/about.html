About AI Bite Detector

AI Bite Detector is an intelligent health risk detection system designed to identify snake bites and insect bites using Artificial Intelligence.

🧠 AI Powered Detection

The system uses a trained deep learning model to analyze bite images and classify them into snake bite, insect bite, or normal skin.

⚠ Risk Level Analysis

After detecting the bite type, the system provides risk levels such as Low, Medium, or High risk to help users understand the severity.

🏥 Emergency Assistance

If a snake bite is detected, the system alerts the user and recommends visiting the nearest hospital immediately.

📍 Nearby Hospital Finder

The application also helps users locate nearby hospitals quickly using integrated map search.
